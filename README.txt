# MeiroGame
Unityを用いたAndroidで動作する迷路ゲーム


プレイの仕方
・BuildAndRunを押して実行する
または
・Buildしてapkファイルを生成する。そのapkファイルをandroid端末にインストールして実行する。
または
・UnityRoomにWebGL版をアップしたのでそれをプレイする。
リンク：
https://unityroom.com/games/meirogame_by_tako


制作人数：一人
制作期間：2018年 8月下旬から9月下旬（初期制作）, 2019年 1月下旬から2月上旬（改良）
制作環境：Unity(Version2018.3.0f2), VisualStudio 2017
使用言語：Unity(C#)
動作環境：Androidを想定


操作方法：
//変更前
スマホ上でフリック操作またはジャイロ操作を行うことでプレイヤーを動かし、スタート地点からゴール地点まで移動する。
・通常プレイは、難易度を選択して無限にプレイできる。
・タイムアタックは難易度順（簡単、普通、難しい）に迷路を解き、そのタイムを競う。

//変更後（現在）
スマホ上でフリック操作によりプレイヤーを動かし、スタート地点からゴール地点まで移動する。
・通常プレイは、体力が０になるまでずっとプレイできる。
　迷路の最短経路でない場所を通ったり、道の途中に設置された罠（赤い床）を踏むと、体力が減少する。
　また、偶に出現する回復床（緑色の床）を踏むと体力が回復する。その際、最短経路でない場所を通っても体力は減少しない。
　体力が０になった瞬間ゲームは終了し、それまでにクリアした迷路の数が記録となる。
・タイムアタックは難易度順（簡単、普通、難しい）に迷路を解き、そのタイムを競う。


迷路生成について：
迷路はランダムに自動生成される。
//変更前
アルゴリズムは穴掘り法　を使用していて、幅優先探索によってスタートとゴールの距離を最も遠くしている。
//変更後（現在）
アルゴリズムは壁伸ばし法を使用していて、幅優先探索によってスタートとゴールの距離を最も遠くしている。
また、スタートからゴールまでの最短経路を求めている。

制作のきっかけ：
自動生成迷路に興味を持ち、習得中であったUnityの勉強も兼ねて、スマホで操作できる暇つぶしになるような迷路ゲームを制作した。
